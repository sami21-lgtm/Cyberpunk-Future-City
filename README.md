🌆 Cyberpunk Portfolio | Sami
A futuristic, cyberpunk-themed personal portfolio carousel featuring 3D PowerPoint-style text animations, neon glowing UI, and smooth transitions. Built with pure HTML, CSS, and Vanilla JavaScript—no frameworks, pure vibes.

CyberpunkHTMLCSSJS

✨ Features
🌪️ 3D PowerPoint Animation: Texts rotate in smoothly from the side using rotateY and translateZ transforms.
💚 Neon UI Design: Custom neon green glowing buttons and cyan navigation arrows matching the dark cyberpunk aesthetic.
🎥 Smooth Carousel: Infinite sliding carousel with auto-play functionality.
🖱️ Interactive Controls: Clickable navigation arrows (< >) and active thumbnail indicators.
📱 Fully Responsive: Optimized perfectly for both desktop and mobile screens.
🛠️ Tech Stack
Frontend: HTML5, CSS3
Scripting: Vanilla JavaScript (No jQuery or React)
Fonts: Orbitron (Google Fonts)
